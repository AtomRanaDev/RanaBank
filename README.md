# 🏦 RanaBank - Banking System

RanaBank is a web-based banking application developed using Java Servlets, JSP, MySQL and Maven.  
It allows users to perform basic banking operations like deposit, withdraw, and transaction tracking.

## 🚀 Technologies Used

- Java Servlets
- JSP
- MySQL
- Maven
- Apache Tomcat
- Railway (Cloud Deployment)

## ✨ Features

- User Signup
- User Login Authentication
- Deposit Money
- Withdraw Money
- Transaction History
- OTP Based Password Reset
- AI Chat Assistant
- Cloud Deployment using Railway

## 📂 Project Structure
src/main/java -> Java Servlets & Backend
src/main/webapp -> JSP Pages
pom.xml -> Maven Dependencies
Dockerfile -> Railway Deployment

## 🗄 Database

MySQL database used with tables:

- users
- transactions
- otp

## 🌐 Deployment

Project deployed using Railway Cloud.
